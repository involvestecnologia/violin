# Button

<Playground>
    <Button onClick={() => alert('Clicked!')} style={{ margin: 5 }}>Default</Button>
    <Button primary onClick={() => alert('Clicked!')} style={{ margin: 5 }}>Primary</Button>
    <Button secondary onClick={() => alert('Clicked!')} style={{ margin: 5 }}>Secondary</Button>
    <Button danger onClick={() => alert('Clicked!')} style={{ margin: 5 }}>Danger</Button>
</Playground>

## Props

<Props of={Button} />

## disabled

Apply native disabled attribute to button.

<Playground>
    <Button disabled style={{ margin: 5 }}>Default</Button>
    <Button primary disabled style={{ margin: 5 }}>Primary</Button>
    <Button secondary disabled style={{ margin: 5 }}>Secondary</Button>
    <Button danger disabled style={{ margin: 5 }}>Danger</Button>
</Playground>

## invert

Change color style for dark background.

<Playground>
    <div style={{ background: '#1F0E3D', padding: 20 }}>
        <Button invert style={{ margin: 5 }}>Default</Button>
        <Button invert primary style={{ margin: 5 }}>Primary</Button>
        <Button invert secondary style={{ margin: 5 }}>Secondary</Button>
        <Button invert danger style={{ margin: 5 }}>Danger</Button>
    </div>
</Playground>

## size

Change style size (height). 

<Playground>
    <Button primary small style={{ margin: 5 }}>Small</Button>
    <Button primary style={{ margin: 5 }}>Normal</Button>
    <Button primary large style={{ margin: 5 }}>Large</Button>
</Playground>

## block

Apply width to 100%.

<Playground>
    <Button primary style={{ margin: 5 }}>Normal</Button>
    <Button primary block style={{ margin: 5 }}>Block</Button>
</Playground>

## icon

Add a icon to left button from [Material Icons library](https://material.io/resources/icons/?style=round).

<Playground>
    <Button secondary small icon="edit" style={{ margin: 5 }}>Small</Button>
    <Button icon="edit" style={{ margin: 5 }}>Normal</Button>
    <Button primary large icon="edit" style={{ margin: 5 }}>Large</Button>
    <br />
    <Button primary small icon="edit" style={{ margin: 5 }} />
    <Button primary icon="edit" style={{ margin: 5 }} />
    <Button primary large icon="edit" style={{ margin: 5 }} />
    <br />
    <Button secondary small icon="edit" style={{ margin: 5 }} />
    <Button secondary icon="edit" style={{ margin: 5 }} />
    <Button secondary large icon="edit" style={{ margin: 5 }} />
</Playground>

## iconRight

Add a icon to right button.

<Playground>
    <Button primary iconRight="arrow_right_alt" style={{ margin: 5 }}>Next step</Button>
</Playground>

## dropdownIcon

Look like `iconRight`, this add a arrow icon to right for dropdown trigger. 

<Playground>
    <Button primary dropdownIcon style={{ margin: 5 }}>Dropdown</Button>
</Playground>

## as 'a' element

As any other component you can change the html element of component with prop `as`.

<Playground>
    <Button as="a" href="http://involves.com" color="primary" style={{ margin: 5 }}>With anchor</Button>
</Playground>
