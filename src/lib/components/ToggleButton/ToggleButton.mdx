# ToggleButton

ToggleButton has two behavior, as radio (default) and cheackbox for select multiple options. 

<Playground>
    <ToggleButton name="option" style={{ margin: 5 }}>1</ToggleButton>
    <ToggleButton name="option" style={{ margin: 5 }}>2</ToggleButton>
    <ToggleButton name="option" style={{ margin: 5 }}>3</ToggleButton>
</Playground>

## Props

<Props of={ToggleButton} />

## multiple

This prop makes possible select multiple itens

<Playground>
    <ToggleButton name="option" multiple style={{ margin: 5 }}>1</ToggleButton>
    <ToggleButton name="option" multiple style={{ margin: 5 }}>2</ToggleButton>
    <ToggleButton name="option" multiple style={{ margin: 5 }}>3</ToggleButton>
</Playground>

## checked

Set checked item by default

<Playground>
    <ToggleButton name="option" checked onChange={() => {}} style={{ margin: 5 }}>1</ToggleButton>
    <ToggleButton name="option" style={{ margin: 5 }}>2</ToggleButton>
    <ToggleButton name="option" style={{ margin: 5 }}>3</ToggleButton>
</Playground>
