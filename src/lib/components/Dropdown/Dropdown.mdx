## preventClose

`preventClose` prop prevent close Dropdown when clicked over or out of component. For this you need manage the `Dropdown` state with `open` prop.

<Playground>
	<DemoDropdown />
</Playground>

```javascript
const DemoDropdown = () => {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <Dropdown preventClose open={isOpen} trigger={<Button primary dropdownIcon onClick={() => setIsOpen(true)}>Click me!</Button>}>
      <div style={{ padding: 10 }}>
        <Checkbox label="Lorem ipsum dolor sit" style={{ padding: '5px 0', display: 'flex' }} />
        <Checkbox label="Duis aute irure dolor in reprehenderit" style={{ padding: '5px 0', display: 'flex' }} />
        <Checkbox label="Excepteur sint occaecat" style={{ padding: '5px 0', marginBottom: 10, display: 'flex' }} />
        <Button block onClick={() => setIsOpen(false)}>Apply</Button>
      </div>
    </Dropdown>
  );
};

export default DemoDropdown;
```

## DropdownItem

`DropdownItem` component is a complementary of `Dropdown` to create a menu list into `Dropdown`.

	</Dropdown>
