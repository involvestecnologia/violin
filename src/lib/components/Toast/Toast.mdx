# Toast

Toast component works with a hook.

<Playground>
  {() => {
    const toast = useToast();
    return (
      <Button onClick={() => toast.add('Lorem ipsum dolor sit amet, consectetur adipiscing elit.')}>Open toast</Button>
    )
  }}
</Playground>

## Usage

Add the provider in your root or App.js file:

```javascript
import { withToastProvider } from '@involves/violin'

const App = () => {
  ...
}

export default withToastProvider(App);
```

Import the hook to use in anywhere of your project.

```javascript
import { useToast } from '@involves/violin'

const MyComponent = () => {
  const toast = useToast();
  render (
    <Button onClick={() => toast.add('My notify message')}>Click to open toast</Button>
  )
}
```

      </Button>

### default options schema

```javascript
...
toast.add('string', {
  timeout: 2000, // number milliseconds
  action: {
    label: null, // string
    method: null // function
  },
  customAction: null, // component/node
  closeButton: false // boolean
}
...
```
