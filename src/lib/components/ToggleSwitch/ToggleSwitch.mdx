# ToggleSwitch

ToggleSwitch is as a checkbox component.

<Playground>
	<State initial={{ check: false }}>
		{({ state, setState }) => (
			<form>
				<ToggleSwitch autoFocus label={state.check ? 'Activated' : 'Deactivated'} checked={state.check} onChange={e => setState({ check: e.target.checked })} />
			</form>
		)}
	</State>
</Playground>

## Props

<Props of={ToggleSwitch} />

> Native props will apply in checkbox input.

## disabled

<Playground>
	<State initial={{ check: true }}>
		{({ state, setState }) => (
