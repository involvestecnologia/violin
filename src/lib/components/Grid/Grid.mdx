# Grid

Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at xs, sm, md, lg & xl viewport widths.

<Playground>
  <Container fluid>
    <Row>
      <Col xs={12} sm={2} md={3} lg={5} xl={4}><Box /></Col>
      <Col xs={12} sm={8} md={6} lg={2} xl={4}><Box /></Col>
      <Col xs={12} sm={2} md={3} lg={5} xl={4}><Box /></Col>
    </Row>
    <Row>
      <Col xs={6}><Box /></Col>
      <Col xs={6}><Box /></Col>
    </Row>
    <Row>
      <Col xs><Box /></Col>
      <Col xs><Box /></Col>
      <Col xs><Box /></Col>
    </Row>
  </Container>
</Playground>
