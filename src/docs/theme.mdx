# Theme

Violin adds a default style with `GlobalStyle` component. It's recommended.

```javascript
import { GlobalStyle } from "violin";

const App = () => {
  return (
    <>
      <GlobalStyle />
      { /** */ }
    </>
  )
}
```

## Custom theme

You can update an specific value from default [theme object](#base-theme-object) or create new properties and values.

```javascript
import { ThemeProvider } from "styled-components";
import { createTheme, theme } from "violin"; 

const myCustomTheme = createTheme({
  palette: {
    primary: theme.palette.purple // This changes primary color only
  },
  customProperty: "some value" // Add new value for use from theme
});

<ThemeProvider theme={myCustomTheme}>
  { /** root component */ }
</ThemeProvider>
```
